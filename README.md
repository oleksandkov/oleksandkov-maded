## 🎯 Repository Purpose

This repository serves as a professional portfolio and resource collection featuring:
- **Professional Website**: Modern, responsive CV website with contact form and project showcase
- **Multi-format Presentations**: Quarto (QMD) presentations rendered to HTML, PDF, and PowerPoint

## 🛠️ Technologies Used

- **Presentation Tools**: Quarto, Reveal.js, PowerPoint
- **Web Technologies**: HTML5, CSS3, JavaScript (Vanilla)
- **Backend**: Node.js with Nodemailer
- **Version Control**: Git & GitHub
- **Development**: VS Code, Quarto CLI

## 🚀 Getting Started

### Quick Start

 **Clone the repository**:

   ```bash
   git clone https://github.com/oleksandkov/oleksandkov-maded.git
   cd oleksandkov-maded
   ```

For full development capabilities:
- See [SETUP.md](docs/SETUP.md) for detailed setup instructions

## 📂 Key Directories

### Presentations (`presentations/`)
Contains technical presentations with multiple output formats:
- **Docker Presentation/**: Interactive Quarto presentation
  - **Docker.qmd**: Complete Docker and Containers presentation
  - Output formats: HTML (RevealJS), PDF, PowerPoint
  - Interactive slides with code highlighting
- **Docker Presentation (Non-Interactive)/**: Web-optimized version for online sharing
- **.github Presentation/**: Presentation about the repository's `.github` folder
- **powerpoint/**: Native PowerPoint files

### Website (`website/`)
Modern, responsive CV website featuring:
- **Responsive Design**: Mobile-first approach, works on all devices
- **Interactive Elements**: Smooth scrolling, animations, form validation
- **Contact Form**: Functional contact form with email integration
- **Project Showcase**: Highlighted projects with technology tags
- **Skills Display**: Organized skill categories
- **SEO Optimized**: Semantic HTML for better search engine visibility
- **Size**: Approximately 73KB total (HTML + CSS + JS)



### Documentation (`docs/`)
Comprehensive guides and references:
- **CONTRIBUTING.md**: How to contribute to the repository
- **SETUP.md**: Development environment setup instructions
- **README.md**: Documentation overview and organization
