### Local Development

1. **Open `index.html`** in your web browser


### Customization

1. **Personal Info**: Edit name, title, and bio in `index.html`
2. **Projects**: Replace example projects with your own
3. **Skills**: Update skill categories and technologies
4. **Colors**: Modify CSS variables in `styles.css`
5. **Contact**: Update contact information and social links

- **GitHub Pages**: Push to GitHub and enable Pages in settings
- **Netlify**: Drag and drop the website folder
